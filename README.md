# CLIO

**Command Line Interface Output**

2016/5/1 by DKZ





a python lib for making a command line interface 

##USE

```py
import clio
print(clio.dotext('CLIoutput\n'))
print(clio.dohr(20))
```

###tree

```
j={
    'code':0,
    'msg':'hello',
    'result':{
        'a':1,
        'b':2,
        'c':[1,2,3],
    }
}
print(clio.dojson(j))
print(clio.dotree(j))

###################output#####################
```
